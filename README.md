# Marouane Baoulla — Portfolio

A premium, production-ready developer portfolio landing page built with **Next.js (App Router)**, **Tailwind CSS**, and **Framer Motion**.

## Features

- Light, modern, premium design (navy + blue/teal gradient)
- Split hero with subtle floating cards and gradient glow
- About, Problem → Solution, Projects, Skills and Contact sections
- Subtle scroll-reveal animations and hover elevation
- Fully responsive, mobile-first layout
- Contact form (opens email client) + email & WhatsApp links

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Customize

- **Contact details**: update `EMAIL` and `WHATSAPP` in `components/Contact.tsx` and footer links in `components/Footer.tsx`.
- **Portrait**: replace the placeholder block in `components/Hero.tsx` with a `next/image`.
- **Projects**: edit the `projects` array in `components/Projects.tsx`.

## Tech Stack

Next.js 14 · React 18 · TypeScript · Tailwind CSS · Framer Motion · lucide-react
